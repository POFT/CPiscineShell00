Exercise 15 : ft_putstr

• Create a function that displays a string of characters on the standard output.
• Here’s how it should be prototyped:

    void ft_putstr(char *str);

Code:
ft_putstr.c

    void    ft_putstr(char *str);
    void    ft_putchar(char c);
    
    void    ft_putstr(char *str)
    {
            int     i;
    
            i = 0;
            while (str[i] != '\0')
            {
                    ft_putchar(str[i]);
                    i++;
            }
    }

ft_putchar.c

    #include <unistd.h>
    
    void    ft_putchar(char c)
    {       
            write(1, &c, 1);
    }
