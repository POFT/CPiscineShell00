Exercise 17 : ft_strcmp

• Reproduce the behavior of the function strcmp (man strcmp).
• Here’s how it should be prototyped:

    int ft_strcmp(char *s1, char *s);

Code:
ft_strcmp.c

    int     ft_strcmp(char *s1, char *s2);
    
    int     ft_strcmp(char *s1, char *s2)
    {
            int     i;
    
            i = 0;
            while ((s1[i] != '\0') && (s2[i] != '\0'))
            {
                    if (s1[i] != s2[i])
                    {
                            return (s1[i] - s2[i]);
                    }
                    i++;
            }
            return (s1[i] - s2[i]);
    }
