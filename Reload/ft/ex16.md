Exercise 16 : ft_strlen

• Reproduce the behavior of the function strlen (man strlen).
• Here’s how it should be prototyped:

    int ft_strlen(char *str);

Code:
ft_strlen.c
    
    int     ft_strlen(char *str);
    
    int     ft_strlen(char *str)
    {
            int     i;
    
            i = 0;
            while (str[i] != '\0')
            {
                    i++;
            }
            return (i);
    }
